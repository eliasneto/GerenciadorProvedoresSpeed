Como funciona

Esta tela usa exatamente a configuracao SMTP carregada no backend do Django. Assim voce consegue validar autenticacao, remetente exibido ao destinatario e entrega antes de usar o envio nas cotacoes.

Cenário recomendado para Microsoft 365: autenticar com a conta SMTP do sistema e enviar com o e-mail da caixa compartilhada no campo De, desde que essa conta tenha permissao Enviar como.
