## Identidade visual em resumo

- **Cor primária**: Amarelo `#FFC107` (destaque, ações, links ativos)
- **Cor de fundo escuro**: Preto `#0F0F0F` (sidebar, botões secundários)
- **Cor de fundo claro**: Cinza-frio `#F8F9FA` (background geral do conteúdo)
- **Fonte**: `Plus Jakarta Sans` — Google Fonts
- **Framework CSS**: Tailwind CSS (CDN)
- **Ícones**: Lucide Icons (CDN)
- **Layout**: Sidebar fixa à esquerda + conteúdo responsivo à direita
