# Modulo Leads

## Objetivo

Gerenciar prospeccao, cotacao e cadastro operacional de empresas/enderecos relacionados ao fluxo comercial inicial.

## Responsabilidades atuais observadas

- fluxo de provedores/cotacao/prospeccao;
- manutencao de empresas e enderecos de lead;
- algumas respostas JSON de apoio a tela;
- parte das rotinas ligadas a lastmile.

## Pontos de atencao

- o mesmo modulo atende multiplos prefixes de URL;
- possui endpoints JSON misturados com navegacao HTML;
- precisa ter contratos externos separados da interface interna.

## Relacao com a iniciativa atual

Pode fornecer dados para API publica futura, mas a exposicao deve ocorrer via camada dedicada de integracoes.
