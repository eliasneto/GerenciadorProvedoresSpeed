# Modulo Core Admin

## Objetivo

Concentrar ferramentas administrativas e operacionais utilizadas pela equipe interna.

## Responsabilidades atuais observadas

- importacoes;
- restauracao de backup;
- testes SMTP;
- rotinas administrativas relacionadas a integracoes.

## Pontos de atencao

- hoje mistura interface administrativa com acionamento de scripts;
- nao deve ser exposto como API publica;
- precisa de fronteira clara em relacao ao novo modulo de integracoes.
