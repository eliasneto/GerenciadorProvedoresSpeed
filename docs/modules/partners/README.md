# Modulo Partners

## Objetivo

Gerenciar parceiros, planos e propostas do fluxo comercial/operacional.

## Responsabilidades atuais observadas

- cadastro e consulta de parceiros;
- gerenciamento de planos;
- criacao e manutencao de propostas;
- controle de status e historico operacional.

## Pontos de atencao

- modulo central para integracoes de consulta;
- possui relacoes com clientes e leads;
- deve ter prioridade alta na definicao de contratos da API v1.
