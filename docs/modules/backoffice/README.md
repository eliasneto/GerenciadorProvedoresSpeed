# Modulo Backoffice

## Objetivo

Executar rotinas operacionais pontuais de apoio ao negocio interno.

## Responsabilidades atuais observadas

- importacoes especificas de cotacao;
- importacoes especificas de atendimentos.

## Pontos de atencao

- escopo pequeno e direcionado;
- deve permanecer interno;
- qualquer necessidade de exposicao externa deve passar pela camada dedicada de API.
