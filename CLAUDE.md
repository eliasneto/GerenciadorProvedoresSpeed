# Instruções para o Claude Code

- Responda sempre em português do Brasil.
- Use uma linguagem clara, objetiva e didática.
- Quando explicar código, explique passo a passo.
- Mantenha nomes de comandos, bibliotecas, frameworks, erros e trechos de código no idioma original quando necessário.
- Ao sugerir comandos, considere que estou usando VS Code e terminal integrado.
- Não responda em inglês, exceto quando for necessário para código, logs, nomes técnicos ou mensagens de erro.
